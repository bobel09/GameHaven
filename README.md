Introduction
Welcome to GameHaven, your ultimate destination for all things gaming! GameHaven is an online game store that provides a seamless experience for users to browse, purchase, and manage their favorite games. This application is built using HTML, CSS, JavaScript, PHP, and utilizes MySQL as its database.

Features
User authentication and registration
Browse and search for games
View detailed game information
Add games to the shopping cart
Checkout and complete purchases
User profile management
Admin panel for managing games and users
Requirements
To run GameHaven on your local machine, ensure you have the following installed:

XAMPP or similar (Apache, MySQL, PHP)
Web browser with JavaScript enabled
Installation
Clone the repository:


git clone [https://github.com/your-username/gamehaven.git](https://github.com/bobel09/GameHaven.git)
Move the project files to your XAMPP's htdocs directory.

Configuration
Start XAMPP and ensure Apache and MySQL servers are running.

Usage
Open your web browser and navigate to http://localhost/index.php.
Register for a new account or log in if you already have one.
Explore the catalog, add games to your cart, and proceed to checkout.
Database
GameHaven uses MySQL as its database. You can find the database schema in the database directory. Import the SQL files to set up the required tables.
